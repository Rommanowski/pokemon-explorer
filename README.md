# Pokemon Explorer

## Uruchomienie

Wymagania: Node.js 20+ oraz npm.

1. Zainstaluj zależności:

   ```bash
   npm install
   ```

2. Uruchom serwer deweloperski:

   ```bash
   npm run dev
   ```

   Aplikacja wystartuje na `http://localhost:5173`.

## Build produkcyjny

```bash
npm run build
npm run preview
```
